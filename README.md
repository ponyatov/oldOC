#  `oldOC`
## custom emLinux for retro/lite computers

(c) Dmitry Ponyatov <<dponyatov@gmail.com>> 2020 MIT

github: https://github.com/ponyatov/oldOC

* monolithic boot image (no packages)
* minimized software
* old hardware support
  * `i386`
    * `qemu386` / `i486sx` `i686`
  * `arm`
    * `rpi3b` / `BCM2837`

*** powered by `metaL`
